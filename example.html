JQuery Checkbox

Checkbox with good visual

 

   

Enable - Disable - Check - Uncheck

 

   

Enable - Disable - Check - Uncheck

 

Using selector jquery for all checkboxes

Enable - Disable - Check - Uncheck

 

Create

$('input:checkbox').checkbox();

// with options
$('input:checkbox').checkbox({
       callback: function()
       {
               alert('Checked: ' + $(this).is(':checked'));
       }
});

Enable

$('input:checkbox').checkbox('enable');

Disable

$('input:checkbox').checkbox('disable');

Check

$('input:checkbox').checkbox('on');

Uncheck

$('input:checkbox').checkbox('off');

 

Rules

- Do not include a parent type "label" for the checkbox element, Instead, use the attribute "for"
